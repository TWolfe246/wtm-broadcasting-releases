How it works

From iRacing to OBS in three pieces.

01

Launcher reads iRacing

The desktop launcher boots a bundled server that reads iRacing's shared memory at 20Hz. No SDK profiles to maintain. No extra installs.

02

Overlays render in OBS

Each overlay is a browser source pointed at the launcher's local server. Add once, leave them in your scene. They turn on and off via the app.

03

You run the broadcast

Race Control tab handles flags, stages, replays, cameras, audio — all without alt-tabbing. Hotkeys for everything. The launcher is the show.

FAQ

Common questions.

Does this work with my OBS scene collection?

Yes. Each overlay is a regular OBS browser source you add to whatever scenes you want. You set it up once and leave it. The launcher controls visibility from the app, so you don't need to switch scenes by hand.

Can I use my own branding / logos / colors?

Yes. Drop your league logo into the branding folder; it shows on the standings and results overlays automatically. Custom car number images and helmet liveries are supported.

What if my internet drops during a race?

Everything runs locally. The launcher talks to iRacing on your machine and serves overlays to OBS on your machine. Internet is only needed for optional features like the season-standings scraper and license checks. License has a 7-day offline grace period — you'll never lose access mid-broadcast.

How are updates handled?

The launcher checks for updates in the background. When a new version is ready, you'll see a banner with release notes. Click "Restart now" or just relaunch the app later — it installs automatically. No reinstalls.

Can I move my license to a different PC?

Yes. In the app's Setup tab, click "Release this machine," then activate on the new PC with the same key. One active machine per license at a time.

Does it work with other sims (rFactor, ACC, etc.)?

iRacing only, at least for now. The composite timing engine is built against iRacing's shared memory format.
